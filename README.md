# Data Storytelling & Statistical Validation

## Project Overview

This project focuses on transforming sales transaction data into actionable business insights through storytelling, dashboard-driven analysis, and statistical validation.

The objective was to communicate business findings effectively and validate key insights using hypothesis testing techniques.

---

## Project Components

### Business Storytelling

A professional business presentation was developed to summarize:

- Business Objectives
- Dataset Overview
- Data Cleaning Process
- KPI Analysis
- Revenue Insights
- Regional Performance
- Customer Analysis
- Strategic Recommendations

### Statistical Validation

Two hypothesis tests were conducted:

#### 1. Independent T-Test

Objective:
Compare average revenue generated across sales channels.

Result:

- T Statistic = -2.408
- P Value = 0.016

Conclusion:

A statistically significant difference exists between sales channels.

#### 2. Chi-Square Test

Objective:
Evaluate the relationship between Payment Method and Sales Channel.

Result:

- Chi-Square = 2.282
- P Value = 0.809

Conclusion:

No statistically significant relationship was found.
